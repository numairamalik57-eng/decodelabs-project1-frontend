# decodelabs-project1-frontend
A responsive frontend interface built with HTML, CSS, and vanilla JavaScript no frameworks, mobile-first design with dark mode and accessibility support.

Responsive Frontend Interface

This is Project 1 for my DecodeLabs Full Stack Developer internship — a responsive frontend interface built with plain HTML, CSS, and JavaScript (no frameworks, as required).

What it does

A simple landing page that adjusts its layout depending on screen size single column on mobile, and a grid layout with a sidebar on desktop. It includes a mobile hamburger menu, a dark/light mode toggle, and nav links that highlight automatically as you scroll through the page.

Built with


HTML5 (semantic tags — header, nav, main, article, aside, footer)
CSS3 (Grid for the overall layout, Flexbox for the nav/buttons, custom properties for theming)
Vanilla JavaScript (DOM events, IntersectionObserver for the scroll highlight)


Accessibility

Added a skip-to-content link, visible focus outlines for keyboard users, and ARIA labels on the interactive buttons, since accessibility was part of the project requirements.

Files


index.html — page structure
style.css — all styling and responsive behavior
script.js — nav toggle, dark mode, and scroll highlighting
README.md — this file


How to run

Just open index.html in a browser. For the best experience (and if you want to test it with Lighthouse), run it through a local server like VS Code's Live Server extension.

Testing
